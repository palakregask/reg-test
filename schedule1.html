Environmental Protection and Emissions Regulations — REG/2020-45

Schedule 1 — List of Controlled Substances

Current to 2025-11-01 · Last amended 2025-08-15 · Previous versions
Note: This Schedule must be read in conjunction with section 11.1(a) of these Regulations. Substances are classified as Class A (highest risk) or Class B (moderate risk) for the purposes of threshold calculations under Schedule 4.

Part I — Greenhouse Gases

Item Substance Chemical Formula Classification Global Warming Potential (100-yr)
1 Carbon Dioxide CO₂ Class B 1
2 Methane CH₄ Class A 28
3 Nitrous Oxide N₂O Class A 265
4 Sulphur Hexafluoride SF₆ Class A 23,500
5 Hydrofluorocarbon-134a HFC-134a Class A 1,300

Part II — Toxic Industrial Compounds

Item Substance Chemical Formula Classification Reportable Threshold (kg/yr)
6 Benzene C₆H₆ Class A 10
7 Toluene C₇H₈ Class B 100
8 Ammonia NH₃ Class B 500
9 Hydrogen Sulphide H₂S Class A 50
10 Formaldehyde HCHO Class A 25

Interpretation

1 Where a substance listed in this Schedule is released in combination with another listed substance, each substance must be reported separately and its individual threshold applied independently.
(2) The Minister may, by order published in the National Gazette, add or remove substances from this Schedule or reclassify a substance between Class A and Class B. Any such order takes effect 90 days after publication unless an earlier date is specified.
